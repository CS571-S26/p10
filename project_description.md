Project Title
Mechanical Gear Simulator and Calculator
Project Description
This project will be a web-based simulator for mechanical gears. The goal is to create an intuitive tool that allows the user to calculate and visualize the relationships between gears of different sizes.

The website will allow users to input parameters such as number of teeth on the driving gear and the driven gear. Based on these values, the application will calculate values like gear ratio, speed ratio, torque relationships, and other relevant features. 

An additional feature will be a visualizer that displays two or more meshing gears based on the input parameters. In addition to the features mentioned, it will display the relationships between the gears based on the values calculated, acting as an educational tool for the user to visualize the values calculated. 

Moreover, we will add educational sections explaining basic gear terminology so that users understand what the different values calculated signify and how they affect other aspects of gear design.

Our target users will be students who are taking introductory engineering courses who want to learn and explore the fundamentals of gear design. The overall goal of the project is to provide an educational tool that will allow users to visualize the different interactions and relationships between gears and to learn more about gear systems.
